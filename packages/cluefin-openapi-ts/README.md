# cluefin-openapi (TypeScript)

TypeScript client for KIS and Kiwoom OpenAPI.

## Install

```bash
bun add cluefin-openapi
# or
npm install cluefin-openapi
```

## Development

```bash
bun install
bun run build
bun run check
bun run typecheck
bun test
```

## Publish

```bash
bun run publish:check
bun run publish:manual
```
